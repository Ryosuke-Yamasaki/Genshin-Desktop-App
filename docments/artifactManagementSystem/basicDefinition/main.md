### テーブル設計

#### Artifacts

| 項目名   | 型             | 必須 | Enum | 制約                          | 説明               |
| -------- | -------------- | ---- | ---- | ----------------------------- | ------------------ |
| id       | `UUID`         | ✅   | ⬜   |                               | 一意な ID          |
| set      | `ArtifactSet`  | ✅   | ⬜   |                               | 聖遺物のセット     |
| type     | `ArtifactType` | ✅   | ✅   |                               | 聖遺物の部位       |
| mainStat | `Stat`         | ✅   | ✅   | type によって選択肢条件あり   | メインオプション   |
| subStats | `Set<Stat>`    | ✅   | ⬜   | 4 つかつ重複不可              | サブオプション     |
| rarity   | `int`          | ✅   | ⬜   |                               | レア度             |
| level    | `int`          | ✅   | ⬜   | rarity によって選択肢条件あり | 聖遺物のレベル     |
| score    | `double`       | ✅   | ⬜   |                               | 聖遺物の会心スコア |
